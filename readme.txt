1 -¿Cuál es la diferencia entre un componente UI y un componente funcional en React?


En React, un componente UI se dedica a la parte visual, enfocándose en mostrar elementos sin involucrar lógica complicada. Por otro lado, un componente funcional es una función de JavaScript que puede incorporar lógica y dinamizar la interfaz de usuario, devolviendo un un elemento react.

2 - ¿Qué son las props en React y cuál es su propósito principal?

Las props en React son objetos que envían información de un componente a otro, permitiendo la comunicación entre ellos. Su función principal es hacer que los componentes sean reutilizables y adaptables, ya que pueden recibir diferentes datos. Además, las props son inmutables, lo que ayuda a mantener el control en la aplicación. En resumen, son esenciales para crear interfaces dinámicas en React.

3- ¿Qué son los children props en React y por qué no se recomienda su uso excesivo?

Los children props en React son una forma de pasar elementos hijos a un componente, lo que permite crear interfaces más flexibles y reutilizables. Aunque son útiles para combinar componentes, usarlos en exceso puede complicar el código, dificultar su mantenimiento y hacer que sea menos claro para otros desarrolladores.

4- ¿Qué es useState en React y para qué se utiliza principalmente?

useState es un hook en React que permite a los componentes funcionales gestionar y almacenar estado. Facilita la actualización automática de la interfaz cuando cambian los valores, siendo útil para manejar interacciones del usuario, como clics o entradas de texto.
